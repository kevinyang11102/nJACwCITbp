# 学生评教管理系统 java359

## 项目概述

学生评教管理系统是基于SSM框架和Vue前端技术的教学评价系统，旨在提高评教效率和准确性，实现对学生评教数据的实时分析与处理。

## 技术栈

### 后端技术

- SSM（Spring + SpringMVC + MyBatis）

### 前端技术

- Vue

## 运行环境

- 开发工具：IDEA
- 运行方式：Tomcat
- 数据库：MySQL 5.7

## 系统角色

系统主要包含以下角色：

- 超级管理员
- 学生用户
- 教师用户

## 功能模块

- 学生评教
- 教师信息管理
- 评教数据统计与分析
- 用户权限管理

## 部署步骤

1. 准备所需环境：IDEA、Tomcat、MySQL 5.7
2. 导入项目代码
3. 创建数据库，执行相应的SQL脚本
4. 配置项目中的数据库连接信息
5. 部署至Tomcat，启动服务
